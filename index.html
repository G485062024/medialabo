第1回 WWW と HTML

Web ページを提供する仕組みのことを WWW といいインターネット上には膨大な数の Web ページが存在する。
また、Web ページを構成する「部品」は大きくHTML,CSS,JavaScript,画像・音声・動画の四つに分けられる。

第2回 CSS による Web ページの装飾

CSSはページを装飾するもので、要素の見た目を変更するものになります。そのためHTMLが最初に与えられて次にその装飾にCSSを記述するという順番になります。CSSをHTMLに適用する方法は三つあり、外部スタイイルシート・内部スタイルシート・インラインスタイル・がありますが今回は外部スタイルシートを使って記述しています。

演習の成果: 演習2-3: Webブラウザ説明ページの装飾
