各授業回の学習事項

第1回 WWW と HTML

Web ページを提供する仕組みのことを WWW といいインターネット上には膨大な数の Web ページが存在する。
また、Web ページを構成する「部品」は大きくHTML,CSS,JavaScript,画像・音声・動画の四つに分けられる。

演習の成果: 演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

CSSはページを装飾するもので、要素の見た目を変更するものになります。そのためHTMLが最初に与えられて次にその装飾にCSSを記述するという順番になります。CSSをHTMLに適用する方法は三つあり、外部スタイイルシート・内部スタイルシート・インラインスタイル・がありますが今回は外部スタイルシートを使って記述しています。

演習の成果: 演習2-3: Webブラウザ説明ページの装飾

第3回 JavaScript の基本

JavaScriptはWeb ページ上で複雑な機能を実現するプログラミング言語でWeb ページを開いた直後や，ボタンを押したときに JavaScript プログラムが実行されます。動更新やアニメーションなど動きのあるページを実現したい時に使われるプログラムです。

演習の成果: 演習3-6:オブジェクトの参照

第4回 DOM と JavaScript

今回は「ボタンを押したらページが変化する」処理を JavaScript プログラムとして記述する方法を学びました。また、DOM（Document Object Model）は Web コンテンツを表現するデータ構造です、DOM を使用することによって、JavaScript などのプログラムによって、Web コンテンツの内容を読み取ったり，コンテンツに変更を加えたりする処理が書けるようになりました。

演習の成果: 演習4-10:足し算ページ

第5回 イベント処理

今回はDOMの高度な機能についての学習を行いました。前回は DOM の機能のうち，検索とテキストの参照，変更，input要素の値取得に絞って学習しましたが、今回は要素の新規作成・要素の編集（要素の属性を追加したり変更したり）・要素の削除などの機能についての学習でした。

演習の成果: 演習5-7:工学部の学科名一覧を表示する
